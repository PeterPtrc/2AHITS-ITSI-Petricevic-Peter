# stdin/stdout

- cat

Hier wird cat genutzt um ein sogenanntes "Echo" zu erstellen.

1. Bsp.
```sh
Eingabe: cat

: (Beliebiges Wort/beliebige Zahl/beliebiger Satz eingeben.) z.B. Beispiel
Ausgabe: Beispiel
         Beispiel

Diesen Effekt nennt man "Echo", da das gleiche Wort 2 mal ausgegeben wird.

mit "Strg + C" kann man wieder aus dem Cat raus
```
- cat >hallo.txt

in dieser Zeile will man im in einem File etwas reinschreiben, z.B. Hallo Welt

```sh
Eingabe: cat >hallo.txt

: (Text/Wort/Zahl Eingabe.)

Ausgabe: (in dem File hallo.txt): Hallo Welt

ACHTUNG: sobald man wieder mit cat in einer Datei reinschreiben will, wird es überschrieben!!!
Um dies zu verhindern. Geben Sie statt ">" das hier ein: ">>". Diese wird dann in die nächte Zeile geschrieben. Hier ein Beispiel.

cat >>hallo.txt
Eingabe: (Zahl/Wort/Satz Eingabe) z.B. Tschau Welt

Ausgabe: Hallo Welt
         Tschau Welt

Auch hier kann man mit "Strg + C" raus.
```

- cat File Ausgabe

In diesem Fall wird cat genutzt um den Inhalt eines Files auszugegeben.

```sh
Eingabe: cat <hallo.txt

Ausgabe: Hallo Welt
         Tschau Welt

So wird mit Hilfe von "cat" der Inhalt eines Files ausgegeben.
```

## Quotes

Quotes werden verwendet um einen Satz mit Leerzeichen, als 1 Sache darzustellen


```sh
Beispiel mkdir

Eingabe: mkdir Urlaubs Fotos
Ausgabe: 2 Ordner wurden erstellt: 1. Urlaubs 2. Fotos. um diese zu verhindern benutzt man 
"..." oder '...'

z.B. 
Eingabe: mkdir "Urlaubs Fotos" oder 'Urlaubs Fotos'

Ausgabe: 1 Ordner wurde erstellt: 1. Urlaubs Fotos

Im Grunde ist es Egal welches von beiden du verwendest. Wenn du jedoch eines der Beiden Zeichen in dem Namen hast dann nimmt man das andere um 1 draus zu machen.

Beispiel:

mkdir "Urlaubs '' Fotos" oder 'Urlaubs "" Fotos'

Hier werden Ordner erstellt mit dem Namen: Urlaubs '' Fotos oder Urlaubs "" Fotos.

Das gleiche gilt auch bei rmdir, usw. 
```

```sh
Beispiel echo

Eingabe: echo >>> Achtung <<<
Ausgabe: bash: syntax error near unexpected token `>`

um diese Eingabe zu korrigieren gibt man auch Quotes ein:

Eingabe: echo ">>> Achtung <<<" oder echo '>>> Achtung <<<'

Ausgabe: >>> Achtung <<<
```

Übung (Textdatei erstellen)

Mit Hilfe von mehreren echo Befehlen können einfache Textdateien erstellt werden.

Erzeuge nur unter Verwendung von echo Befehlen die Datei made_by_echoing.txt mit Inhalt:

=====================
=    HTL BRAUNAU    =
=====================
= 2AHITS Gruppe n   =
= x Schülerinnen    =
= y Schüler         =
=====================

Setze für n, x und y die richtigen Werte ein.

```sh
Eingabe: echo "=====================" > made_by_echoing.txt 
         echo "=    HTL BRAUNAU    =" >> made_by_echoing.txt
         echo "=====================" >> made_by_echoing.txt
         echo "= 2AHITS Gruppe 2   =" >> made_by_echoing.txt
         echo "= 0 Schülerinnen    =" >> made_by_echoing.txt
         echo "=26 Schüler         =" >> made_by_echoing.txt
         echo "=====================" >> made_by_echoing.txt

Ausgabe: =====================
         =    HTL BRAUNAU    =
         =====================
         = 2AHITS Gruppe 2   =
         = 0 Schülerinnen    =
         =26 Schüler         =
         =====================
```

Übung (C Hallo Welt)

Erstelle nur mit Hilfe von echo Befehlen eine Datei hello.cpp mit folgendem Inhalt:

#include <iostream>

int main() {
  printf("\n\t*** Hallo Welt ***\n");
  return 0;
}

Hinweis: Verwende einfache Hochkommas rund um den mit echo auszugebenden Text (Beispiel: echo 'text') ansonsten hat die shell Schwierigkeiten mit den Sonderzeichen #,<,> etc.

Compiliere das C Programm mit folgendem Befehl in der Kommandozeile:

g++ -o hello hello.cpp

Das Programm kann so gestartet werden:

./hello

Mit Strich-Punkt (;) getrennt lassen sich auch 2 Befehle hintereinander inn der Kommandozeile schreiben – probiere auch das einmal aus:

g++ -o hello hello.cpp;./hello

```sh
Eingabe: echo '#include <iostream>' > hello.cpp
         echo '' >> hello.cpp
         echo 'int main() {' >> hello.cpp
         echo '  printf("\n\t*** Hallo Welt ***\n");' >> hello.cpp
         echo '  return 0;' >> hello.cpp
         echo '}' >> hello.cpp

Ausgabe: #include <iostream>

         int main() {
         printf("\n\t*** Hallo Welt ***\n");
         return 0;
         }

```
