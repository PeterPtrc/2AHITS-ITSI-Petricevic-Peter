# Erklärung
```sh
head: Zeigt die ersten Zeilen einer Datei oder Ausgabe an (normal: 10). Man benutzt es oft, um schnell den Anfang einer Datei anzuschauen. Mit -n kann man die Anzahl der Zeilen festlegen.

grep: Sucht nach einem bestimmten Text oder Muster in Dateien oder Ausgaben. Es zeigt nur die Zeilen an, die zum Suchbegriff passen. 

tr: Verändert Zeichen in einem Satz/Wort usw. Man kann damit z.B. Buchstaben ersetzen, löschen oder Kleinbuchstaben in Großbuchstaben umwandeln. 

shuf: Mischt Zeilen zufällig. Das wird oft benutzt, um zufällige Reihenfolgen zu erzeugen, z. B. bei Listen oder Zahlen. Man kann auch nur einige zufällige Zeilen auswählen.

seq: Erzeugt eine Folge von Zahlen. Zum Beispiel gibt seq 1 5 die Zahlen von 1 bis 5 aus. Man kann auch Start, Ende und Schrittweite angeben.

date: Zeigt das aktuelle Datum und die Uhrzeit an. Mit Formatoptionen kann man bestimmen, wie die Ausgabe aussieht. Wird oft in Skripten verwendet.

ROT13: ROT13 ist eine Verschlüsselung, bei der jeder Buchstabe im Alphabet um 13 Stellen verschoben wird (A → N, B → O usw.). Wendet man ROT13 wieder an, erhält man wieder den vorherigen Text.

tail: tail zeigt die letzten Zeilen einer Datei oder Ausgabe (standardmäßig 10). Es wird oft verwendet, um das Ende von Logdateien zu sehen. Mit -n kann man die Anzahl der Zeilen ändern.
```
