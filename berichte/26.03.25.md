Mit chmod +x pwcheck -> machst du die Datei startbar. Danach kannst du sie mit ./pwcheck ausführen. Das Programm checkt, wie sicher ein Passwort ist.
Mit xxd pwcheck -> kannst du dir anschauen, wie die Datei „innen drin“ aussieht (in Hex-Zahlen und Zeichen).

# gdb kurz erklärt
Mit gdb pwcheck -> startest du ein Tool, mit dem du das Programm Schritt für Schritt beobachten kannst. Beenden geht mit q.

# Was macht ihr in der Übung?
Ihr schaut euch an, wie das Programm funktioniert (Reverse Engineering). Ziel ist es zu verstehen, wie die Passwortprüfung abläuft – und dann einen Weg zu finden, sie zu umgehen.

# Wichtige Befehle:

- b main -> Startpunkt festlegen
- run -> Programm starten
- disas main -> Code anschauen
- c -> weiterlaufen lassen
- si / ni -> Schritt für Schritt durchgehen
- info registers eflags -> wichtige Infos zum Zustand sehen

Der Rest ist sich zeitlich nicht mehr ausgegangen.
