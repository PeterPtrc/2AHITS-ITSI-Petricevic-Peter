## Aufgabe 3

Informiere dich über die Befehle rm, rmdir, cp und mv. Probiere die Befehle aus und dokumentiere deine Erkenntnisse im Arbeitsbericht.

```sh
(rm (remove))

rm first_dir

Löscht Dateien und Verzeichnisse.
```
```sh
(rmdir (remove directory))

rmdir first_dir

Löscht leere Verzeichnisse.
```
```sh
(cp (copy))

cp abcd.01.1.txt copy.01.1.txt
(kopiert abcd.01.1.txt mit dem Namen copy.01.1.txt)

cp abcd.01.1.txt img/
(kopiert abcd.01.1.txt in dem Ordner img)

(cp) wird standartsmäßig nur für dateien benutzt, um Verzeichnisse zu kopieren, benutze ein 
-r oder -R (rekursiv)

cp -r first_dir berichte
```

```sh
(mv (move))
man kann eine Datei verschieben

mv neu.txt second_dir

Verschiebt die Datei neu in den Ordner second_dir

mv neu.txt umbenannt.txt
So kann man Sachen umbenennen
```
